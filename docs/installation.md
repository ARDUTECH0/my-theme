# 📥 التثبيت والإعداد | Installation & Setup

> [!NOTE]
> هذا الدليل يشرح خطوات تثبيت قالب ECM وإعداده لأول مرة.

---

## 📦 الخطوة 1: التحميل والرفع

### الطريقة أ: الرفع من لوحة التحكم (موصى بها)

1. حمّل ملف القالب `my-theme.zip`
2. افتح لوحة تحكم ووردبريس
3. انتقل إلى: **مظهر** ← **قوالب** ← **أضف جديد** ← **رفع قالب**
4. اختر ملف `my-theme.zip` واضغط **التنصيب الآن**

```
Dashboard → Appearance → Themes → Add New → Upload Theme
```

### الطريقة ب: الرفع عبر FTP

1. فُك ضغط ملف `my-theme.zip` على جهازك
2. ارفع مجلد `my-theme/` إلى المسار التالي على السيرفر:

```
/wp-content/themes/my-theme/
```

3. تأكد أن هيكل الملفات صحيح:

```
wp-content/
└── themes/
    └── my-theme/
        ├── style.css
        ├── functions.php
        ├── header.php
        ├── footer.php
        ├── screenshot.png
        └── ...
```

> [!WARNING]
> تأكد من رفع **المجلد الداخلي** وليس مجلد مُغلَّف إضافي. يجب أن يكون ملف `style.css` مباشرةً داخل `my-theme/`.

## ⚡ الخطوة 2: تفعيل القالب

1. انتقل إلى: **مظهر** ← **قوالب**
2. ابحث عن قالب **ECM — E.Camera.Man**
3. اضغط **تفعيل**

```
Dashboard → Appearance → Themes → ECM — E.Camera.Man → Activate
```

بعد التفعيل، ستلاحظ تغيير تصميم الموقع فوراً إلى التصميم الداكن الاحترافي.

---

## 🎯 الخطوة 3: قائمة الإعداد الأولي

بعد التفعيل، اتبع هذه القائمة لإعداد القالب بالكامل:

### 🏷️ أ. الشعار والهوية

| الإعداد | المسار | الشرح |
|---------|--------|-------|
| شعار الموقع | تخصيص ← هوية الموقع | ارفع شعار بصيغة PNG أو SVG |
| عنوان الموقع | تخصيص ← هوية الموقع | اسم الموقع (ECM) |
| وصف الموقع | تخصيص ← هوية الموقع | سطر وصف مختصر |
| أيقونة الموقع | تخصيص ← هوية الموقع | Favicon بحجم 512×512 |

### 🎨 ب. الألوان

| الإعداد | المتغير | القيمة الافتراضية | الشرح |
|---------|---------|------------------|-------|
| اللون الأخضر المميز | `ecm_green_color` | `#00e676` | اللون الأساسي للعلامة |
| خلفية الصفحة | `ecm_bg_deep` | `#0a0a0a` | لون الخلفية الرئيسي |
| خلفية البطاقات | `ecm_bg_card` | `#111111` | لون خلفية البطاقات |

```
Dashboard → Appearance → Customize → ألوان الثيم
```

### 📌 ج. القوائم

1. انتقل إلى: **مظهر** ← **قوائم**
2. أنشئ قائمة جديدة
3. عيّن الموقع إلى **primary** (القائمة الرئيسية)

```
Dashboard → Appearance → Menus
Create → Assign to "primary" location
```

### 📱 د. بيانات التواصل

| الإعداد | المتغير | الشرح |
|---------|---------|-------|
| واتساب | `ecm_whatsapp` | رقم الواتساب بالصيغة الدولية |
| البريد | `ecm_email` | عنوان البريد الإلكتروني |
| إنستغرام | `ecm_instagram` | رابط حساب إنستغرام |
| يوتيوب | `ecm_youtube` | رابط قناة يوتيوب |

```
Dashboard → Appearance → Customize → بيانات التواصل
```

### 🦶 هـ. إعدادات الفوتر

| الإعداد | المتغير | الشرح |
|---------|---------|-------|
| نص الحقوق | `ecm_footer_copy` | نص حقوق النشر |
| الوصف المصغّر | `ecm_footer_tagline` | سطر وصف صغير في الفوتر |

```
Dashboard → Appearance → Customize → إعدادات الفوتر
```

## 🔌 الخطوة 4: الإضافات المطلوبة والموصى بها

### إضافات موصى بها بشدة

| الإضافة | الحالة | السبب |
|---------|--------|-------|
| **Elementor** | ⭐ موصى بها بشدة | مطلوبة لاستخدام ودجات ECM المخصصة الخمس |
| **Elementor Pro** | 🔸 اختياري | ميزات إضافية مثل Theme Builder |

### إضافات اختيارية مفيدة

| الإضافة | الحالة | السبب |
|---------|--------|-------|
| **ACF (Advanced Custom Fields)** | 🔸 اختياري | لإضافة حقول مخصصة متقدمة |
| **WPForms Lite** | 🔸 اختياري | لإنشاء نماذج تواصل |
| **Yoast SEO** | 🔸 اختياري | لتحسين محركات البحث |
| **WP Super Cache** | 🔸 اختياري | لتحسين أداء الموقع |

### تثبيت Elementor

```
Dashboard → Plugins → Add New → Search "Elementor" → Install → Activate
```

```
لوحة التحكم ← إضافات ← أضف جديد ← ابحث عن "Elementor" ← تنصيب ← تفعيل
```

> [!TIP]
> بعد تثبيت Elementor، ستظهر ودجات ECM المخصصة تلقائياً في قسم **ECM Widgets** داخل محرر Elementor.
