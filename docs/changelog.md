## 🟢 v2.0.0 — الإصدار الحالي
_التاريخ: يونيو 2026_

### ✨ جديد
- **صفحة رئيسية جاهزة** (`front-page.php`) — تظهر فوراً عند تفعيل الثيم بكل الأقسام
- **قوالب كاملة**: `single.php`, `archive.php`, `search.php`, `404.php`, `sidebar.php`, `comments.php`

### 🔧 تحسينات
- إعادة كتابة كاملة لـ `style.css` — **26 قسم منظم** + Design Tokens
- إلغاء الاعتماد على **jQuery** في الـ frontend
- **IntersectionObserver** لتأثيرات fade-in عند التمرير
- **Navbar shrink** عند التمرير بـ requestAnimationFrame
- **Overlay backdrop** + إغلاق بـ Escape key للقائمة
- توسيع **Customizer**: ألوان خلفية + Social Links + Footer Tagline
- تحسين **Elementor Widgets**: controls أكثر + Style tab أقوى
- **Admin bar fix** — النافيبار يتوافق مع شريط WordPress
- **Sticky post badge** — شارة "مثبّت" على البوستات المثبتة

### 🗑 محذوف

- اعتماد jQuery في ecm-theme.js
